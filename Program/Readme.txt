Cách chạy chương trình :
- Chọn file exe tương ứng với máy tính bạn đang sử dụng (32-bit/64-bit)
- Đặt file exe và thư mục train vào cùng một thư mục
- Mở chương trình và chọn thao tác :
+ 0 : khởi tạo lại file metadata từ dữ liệu trong thư mục train (tuyệt đối không được đổi tên)
+ 1 : tìm kiếm với từ khóa tiếng việt
+ 2 : thêm một file vào metadata (nhập đường dẫn tuyệt đối tới bất cứ file nào trong máy tính theo dạng chuẩn, chỉ chứa các kí tự ascii)
    => nếu file này không tồn tại hoặc đã được thêm vào metadata thì chương trình sẽ báo lỗi
+ 3 : xóa một file khỏi metadata (nhập đường dẫn tuyệt đối tới bất cứ file nào trong máy tính theo dạng chuẩn, chỉ chứa các kí tự ascii)
    => nếu file này không tồn tại hoặc chưa được thêm vào metadata thì chương trình sẽ báo lỗi
+ 4 : kết thúc chương trình
